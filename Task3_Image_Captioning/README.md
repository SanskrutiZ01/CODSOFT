# Image Captioning AI

## CodSoft AI Internship – Task 3

This project generates captions for images using Artificial Intelligence.

It combines Computer Vision and Natural Language Processing to analyze an image and automatically generate a caption.

## Features

- Generates captions for images
- Displays image with caption
- Uses pre-trained BLIP model
- Beginner-friendly AI project

## Technologies Used

- Python
- Transformers
- Pillow
- Matplotlib

## How to Run

```bash
python Task3_Image_Captioning/image_caption.py
```

## Example Output

```text
Generated Caption:
A Dog Running In The Grass With A Red Ball In Its Mouth
```
