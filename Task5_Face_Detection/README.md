# Face Detection AI

## CodSoft AI Internship – Task 5

This project detects human faces in images using Artificial Intelligence.

It uses OpenCV’s Haar Cascade classifier to identify faces and draw rectangles around them automatically.

## Features

- Detects faces in images
- Draws rectangles around detected faces
- Shows total number of faces
- Displays output image
- Beginner-friendly AI project

## Technologies Used

- Python
- OpenCV
- Haar Cascade Classifier

## How to Run

```bash
python Task5_Face_Detection/face_detection.py
```

## Example Output

```text
Faces detected: 3
```

Output window:

- Image displayed
- Green rectangles around detected faces
