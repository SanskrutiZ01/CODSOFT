# Movie & Anime Recommendation System

## CodSoft AI Internship – Task 4

This project is a simple recommendation system built using Python as part of the CodSoft Artificial Intelligence Internship.

It recommends movies or anime based on the user’s preferences such as industry and genre.

## Features

- Recommends movies and anime
- User selects preferred industry
- User selects preferred genre
- Content-based filtering
- Beginner-friendly Python project

## Supported Industries

- Hollywood
- Bollywood
- Korean
- Japanese
- Chinese
- Thai

## Supported Genres

- Action
- Romance
- Drama
- Thriller
- Comedy
- Sci-Fi
- Anime

## Technologies Used

- Python
- CSV

## How to Run

```bash
python Task4_Recommendation_System/recommendation.py
```
