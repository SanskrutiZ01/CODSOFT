# Tic-Tac-Toe AI

## CodSoft AI Internship – Task 2

This project is a Tic-Tac-Toe game developed in Python as part of the CodSoft Artificial Intelligence Internship.

The player competes against an AI opponent that uses the **Minimax algorithm** to choose the best move.

## Features

- Human vs AI gameplay
- Minimax algorithm
- Win / draw detection
- Input validation
- Play again option
- Console-based interface

## Technologies Used

- Python

## How to Run

```bash
python tic_tac_toe.py
```

## Example Board Positions

```text
1 | 2 | 3
4 | 5 | 6
7 | 8 | 9
```
